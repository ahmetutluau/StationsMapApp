# StationsMapApp
An application where I show the stations data, which is refreshed every 30 seconds, on the map, and you can click on the stations markers and see the station data in more detail.

## Technologies
MVVM,
Singleton Design Patern,
Reactive Programing,
Unit Tests,
UIKit,
UrlSession,
Generics
